# Конфигурация для русской клавиатуры

Конфигурация LazyVimx с поддержкой русской раскладки клавиатуры.

## Что включено

- ✅ **langmapper** - Трансляция русской раскладки клавиатуры
- ✅ Улучшенные движения (работают с русской раскладкой)
- ✅ Основные переопределения
- ✅ Рекомендуемые UI extras

## Ключевая функция: langmapper

Extra `motions.langmapper` позволяет использовать Vim движения без переключения с русской раскладки.

### Как это работает

Langmapper автоматически транслирует русские клавиши в английские эквиваленты:

| Русская | Английская | Действие          |
| ------- | ---------- | ----------------- |
| `ц`     | `w`        | Следующее слово   |
| `и`     | `b`        | Предыдущее слово  |
| `ш`     | `i`        | Режим вставки     |
| `л`     | `k`        | Вверх             |
| `о`     | `j`        | Вниз              |
| `р`     | `h`        | Влево             |
| `д`     | `l`        | Вправо            |

### Leader клавиша

Leader клавиша работает с обеими раскладками:
- Английская: `<Space>ff` - Найти файлы
- Русская: `<Space>аа` - Найти файлы (то же действие)

### Интеграция с Which-Key

Подсказки which-key автоматически показывают русские клавиши при активной раскладке.

## Установка

1. Скопируйте `init.lua` в `~/.config/nvim/init.lua`
2. Запустите Neovim
3. Используйте Vim движения с русской раскладкой - просто работает!

## Поддерживаемые функции

### Все Vim движения
- `hjkl` навигация (рлод)
- `w`, `b`, `e` движения (ц, и, у)
- `f`, `t` найти/до (а, е)
- `d`, `c`, `y` операции (в, с, н)

### Горячие клавиши LazyVimx
- Leader маппинги (`<Space>...`)
- Spider движения
- TreeWalker навигация
- Все кастомные горячие клавиши

### Исключено из трансляции
- Режим вставки (печатайте русский текст как обычно)
- Командный режим (`:` команды)
- Режим терминала

## Советы

1. **Не переключайте раскладку**: Просто используйте русскую раскладку для всего
2. **Выучите позиции русских клавиш**: Через некоторое время появляется мышечная память
3. **Используйте which-key**: Нажмите leader и увидите доступные команды на русском

## Тестирование

Проверьте, работает ли langmapper:

```vim
:lua print(require("langmapper").get_lang())
```

Должна показаться текущая раскладка.

## Решение проблем

### Движения не работают с русской раскладкой

1. Проверьте, что langmapper загружен:
   ```vim
   :LazyExtras
   ```
   Найдите `motions.langmapper` - должен быть включён

2. Проверьте маппинг:
   ```vim
   :verbose map ц
   ```
   Должен показать маппинг

3. Перезапустите Neovim после включения extra

### Which-key показывает только английские клавиши

Интеграция langmapper + which-key требует загрузки обоих плагинов. Проверьте:

```vim
:Lazy
```

Оба плагина `langmapper.nvim` и `which-key.nvim` должны присутствовать.

## Альтернатива: Умное переключение раскладки

Если предпочитаете автоматическое переключение раскладки:

**macOS**: Используйте [Punto Switcher](https://yandex.ru/soft/punto/)
**Linux**: Используйте [xkb-switch](https://github.com/grwlf/xkb-switch)
**Windows**: Используйте [Vulture](https://github.com/RavenX8/Vulture)

Но langmapper рекомендуется для бесшовного опыта!
