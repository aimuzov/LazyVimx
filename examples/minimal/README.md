# Минимальная конфигурация LazyVimx

Минимальная настройка с только переопределениями плагинов, без extras.

## Что включено

- ✅ Базовый LazyVim
- ✅ Переопределения плагинов LazyVimx (33 модуля)
- ❌ Нет extras

## Сценарий использования

Идеально для пользователей, которые:
- Хотят LazyVim с небольшими улучшениями
- Предпочитают выбирать extras индивидуально позже
- Хотят максимальный контроль над тем, что загружается

## Установка

1. Скопируйте `init.lua` в `~/.config/nvim/init.lua`
2. Запустите Neovim
3. Всё установится автоматически!

## Следующие шаги

Включайте extras через `:LazyExtras` или добавляйте их в конфиг:

```lua
{ import = "lazyvimx.extras.ui.better-diagnostic" },
{ import = "lazyvimx.extras.motions.langmapper" },
```

## Производительность

- **Время запуска**: ~55-85ms (минимальные накладные расходы)
- **Количество плагинов**: ~60-70 (база LazyVim + необходимые плагины)
