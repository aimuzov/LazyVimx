Полный опыт LazyVimx со всеми 48 extras включёнными.

## Что включено

- ✅ Базовый LazyVim
- ✅ Все 48 extras LazyVimx
- ✅ Все 33 переопределения плагинов
- ✅ Кастомные горячие клавиши
- ✅ Улучшенный UI
- ✅ Функции продуктивности

## Сценарий использования

Идеально для пользователей, которые:
- Хотят полный опыт LazyVimx
- Доверяют подобранному набору функций
- Хотят максимальную продуктивность из коробки

## Установка

1. Скопируйте `init.lua` в `~/.config/nvim/init.lua`
2. Запустите Neovim
3. Дождитесь завершения установки
4. Перезапустите Neovim

## Конфигурация

Настройте в `opts` плагина или создайте `~/.config/nvim/lua/plugins/lazyvimx.lua`:

```lua
return {
  "aimuzov/LazyVimx",
  opts = {
    colorscheme = "tokyonight",
    bufferline_groups = {
      ["Components"] = "components/",
      ["Pages"] = "pages/",
    },
  },
}
```

## Производительность

- **Время запуска**: ~80-120ms (+30-40ms по сравнению с минимальной)
- **Количество плагинов**: ~100-120
- **Память**: Умеренная (с включёнными extras очистки)

## Отключение функций

Не нужно всё? Отключайте отдельные extras:

```lua
-- Вместо core.all используйте выборочные импорты
{ import = "lazyvimx.extras.core.overrides" },
{ import = "lazyvimx.extras.ui.better-diagnostic" },
{ import = "lazyvimx.extras.motions.langmapper" },
```
